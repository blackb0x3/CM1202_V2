Python Basics

Printing to the Screen

The print() function is used to print a value to the screen (called the 'console'):

  • Example: print("Hello World!") will output "Hello World!" to the console.
  • Example: print(2 + 4) will output 6 to the console (since 2 + 4 = 6).

Variables, Data Types and User Input

Variables in programming are places where data can be stored and accessed later while a program is running.

In many programming languages, a data type must be specified before the variable is declared.

There are many data types:

  • Strings (i.e. text)
  • Integers (i.e. whole numbers)
  • Booleans (a special type of value which can only be either 'True' or 'False')
  • Floating Point Numbers (i.e. decimal numbers)
  • Lists (special variables that can hold more than one value at a time, unlike other variables)

In Python, the data type doesn't need to be specified.

This means declaring a variable is easy! Simply type the name of the variable you want to create, and set it to some value using the '=' operator.

  • Example: name = "Luke Barker"
  • Example: age = 20
  • Example: isHungry = True
  • Example: percentageScore = 76.4
  • Example: myList = [name, age, isHungry, percentageScore]

Note: print(myList) will show ['Luke Barker', 20, True, 76.4] on the console.


User input can be retrieved using the input() function.

This function can have a text value inside of it (called an argument) that allows text to be printed which can help prompt a user for a value.

For Example: input("Enter your name: ") will output Enter your name: |

Strings in Python

Strings are amongst the most popular types in Python. We can create them simply by enclosing characters in quotes.

To access substrings (parts of a string), use the square brackets for slicing the original string along with the index or indices of the string you want to select.

For example:
string = "Hello World!"
print("string[0]: " + string[0])
print("string[2:7]: " + string[2:7])
print("string[4:]: " + string[4:])
print("string[:8]: " + string[:8])

The above code will print out:
string[0]: H
string[2:7]: llo W
string[4:]: o World!
string[:8]: Hello Wo

Program Flow

In programming, you can control where the next line of code is executed.

In Python, one way where this can be achieved is with if, elif and else statements.

To create one of these statements, you need to use a conditional operator:

OperatorMeaning
==is equal to
< is less than
>is greater than
<=is less than or equal to
>=is greater than or equal to
!not (used in conjunction with other conditional operators, e.g. "!=" for "not equal to")

For example:
age = input("How old are you? ")

if age <= 30:
print("You're looking good!")
elif age >= 60:
print("You're getting old!")
else:
print("You're not too bad!")
