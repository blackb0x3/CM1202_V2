The "Pascaline"

1642: Blaise Pascal

  • First mechanical machine to perform calculations.
  • Works like a cars odometer, but the wheels are rotated manually.
  • Higher wheel rotates by one when a lower dial position produces a carry.
    • Can do addition directly on the machine.
    • Can do subtraction using an algorithm called "complements to 9".

The Difference Engine

1822: Charles Babbage (original idea by J.H. M&uu;mlller, 1786)

  • Evaluates Polynomials (e.g. trigonometric functions, logarithms etc.) using series expansion.

The ENIAC (Electronic Numerical Integrator And Calculator)

1943 - 1946

  • Operational in 1946.
  • World's first fully electronic computer.
  • 300 multiplications or 5000 additions per second.
  • Numerous components followed on from this device (e.g. transistors, vacuum tubes, crystal diodes etc.)
  • Couldn't yet store digital programs in memory.

The Small Scale Experimental Machine (SSEM)

1948: Tom Kilburn & Freddie Williams

  • The world's first stored-program electronic computer.
    • Stored the values from calculations and program instructions in the same memory space.
  • This was faster than ENIAC, since performing a different function required a reconfiguration of the electronic circuitry.

Modern Computers

Contain a variety of new components compared to ENIAC and SSEM

  • Processor - Executes instructions and stores results in cache and RAM
  • RAM (Random Access Memory) - Temporary storage device used to store values and instructions waiting to be executed by the processor.
  • Hard Drive - Form of permanent data storage, used to store program files and data, such as those for a computer's operating system.
  • BIOS - Performs a POST (Power On Self Test) on the computer to make sure that the components are functioning as they should. It also contains a program which attempts to locate a bootable operating system from the hard drive and loads this into the RAM.
  • Expansion Cards - Used to add extra functionality to a computer; such as outputting images and sound or enabling network communications between other devices.
  • Peripherals - Devices which can be connected to a computer using cables or wireless signals.
    • Inputs - Keyboard, Mouse, Microphone etc.
    • Outputs - Monitor, Speakers, Printer
    • Can you think of any devices that could be classed as both an input and output device?
